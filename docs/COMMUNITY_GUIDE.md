# 시민법정 커뮤니티 운영 가이드

## 개요

이 가이드는 시민법정 커뮤니티(카카오톡 오픈채팅방, 텔레그램)에서 민주적 의사결정을 진행하는 방법을 설명합니다.

참여자들이 직접 제안하고, 투표하고, 운영 규칙을 스스로 만들어가는 것이 목표입니다.

**커뮤니티 규칙**: https://communityrule.info/create/?r=1767161408702

---

## 1. 카카오톡 오픈채팅방 운영 방법

카카오톡은 봇 자동화가 불가능하므로 **수동**으로 운영합니다.

### 1.1 제안하기

누구나 아래 형식으로 제안할 수 있습니다:

```
#제안 [제안 내용]
```

**예시:**
```
#제안 월례회의를 매주 토요일 오후 3시로 변경하자
#제안 신규 회원 환영 메시지를 추가하자
#제안 커뮤니티 규칙에 '상호 존중' 조항을 추가하자
```

### 1.2 투표 만들기 (방장/관리자)

제안이 올라오면 방장이 투표를 생성합니다:

1. **채팅방 메뉴** (우상단 ≡) 클릭
2. **투표** 선택
3. 투표 내용 입력:
   - **제목**: `[제안] 제안 내용`
   - **선택지**:
     - 찬성
     - 반대
     - 기권
   - **마감일**: 24시간 후 (또는 적절한 기간)
   - **복수 선택**: 끄기
   - **익명 투표**: 끄기 (공개 투표 권장)
4. **완료** 클릭

### 1.3 투표 결과 공지 (방장/관리자)

투표가 마감되면 결과를 공지합니다:

```
📊 투표 결과 발표

📝 제안: [제안 내용]
👤 제안자: [이름]

✅ 결과: 통과 / 부결

📈 투표 현황:
  찬성: OO표
  반대: OO표
  기권: OO표
  총 참여: OO명

[통과 시] 커뮤니티 규칙에 반영됩니다.
```

### 1.4 규칙 반영

통과된 제안은 CommunityRule에 반영합니다:
1. https://communityrule.info/create/?r=1767161408702 접속
2. 해당 규칙 수정/추가
3. 저장 후 링크 공유

## 2. 텔레그램 운영 방법 (자동화)

텔레그램은 봇이 자동으로 처리합니다.

### 2.1 제안하기

그룹에서 아래 형식으로 메시지를 보내면 **자동으로 투표가 생성**됩니다:

```
#제안 [제안 내용]
```

**예시:**
```
#제안 월례회의를 매주 토요일 오후 3시로 변경하자
```

### 2.2 투표 기간 조정 (선택사항)

제안자가 직접 투표 기간을 설정할 수 있습니다:

```
#제안 48시간 [제안 내용]
#제안 7일 [제안 내용]
```

**기간 설정 규칙:**
- **기본값**: 24시간 (기간 미지정 시)
- **시간 단위**: 1~240시간 (예: `#제안 48시간 내용`)
- **일 단위**: 1~10일 (예: `#제안 3일 내용`)

**예시:**
```
#제안 48시간 긴급 안건에 대해 투표합니다
#제안 7일 중요한 규칙 변경 제안입니다
```

### 2.3 자동 처리 흐름

1. **#제안 감지**: 봇이 `#제안`으로 시작하는 메시지 감지
2. **알림 전송**: "새로운 제안이 등록되었습니다!" 메시지 전송
3. **투표 생성**: 찬성/반대/기권 투표 자동 생성 (지정 기간 또는 기본 24시간)
4. **결과 발표**: 마감 후 자동으로 결과 공지
5. **기록 저장**: 모든 제안과 결과가 데이터베이스에 저장됨

### 2.4 투표 참여

생성된 투표에서 선택지를 클릭하면 됩니다:
- ✅ 찬성
- ❌ 반대
- ⏸️ 기권

## 3. 민주적 의사결정 원칙

### 3.1 투표 규칙

| 항목 | 내용 |
|------|------|
| 투표 기간 | 기본 24시간 (제안자가 1시간~10일 범위 내 조정 가능) |
| 통과 기준 | 찬성 > 반대 (기권 제외) |
| 동률 처리 | 부결 |
| 최소 참여 | 없음 (추후 조정 가능) |

### 3.2 제안 가이드라인

**좋은 제안:**
- 구체적이고 실행 가능한 내용
- 커뮤니티 전체에 영향을 주는 사안
- 명확한 변경 사항 제시

**피해야 할 제안:**
- 개인적인 요청
- 모호하거나 추상적인 내용
- 이미 진행 중인 동일 제안

### 3.3 토론

- 투표 전 충분한 토론을 권장합니다
- 서로 다른 의견을 존중해주세요
- 질문이 있으면 제안자에게 물어보세요

## 4. CommunityRule 활용

CommunityRule은 커뮤니티 규칙을 시각적으로 정리하는 도구입니다.

**현재 규칙 보기**: https://communityrule.info/create/?r=1767161408702

### 4.1 규칙 구성 요소

- **Culture (문화)**: 커뮤니티의 가치와 규범
- **Decision (의사결정)**: 투표 방식, 합의 방법
- **Process (절차)**: 가입, 제명, 분쟁 해결 등
- **Structure (구조)**: 역할, 위원회, 조직 구조

### 4.2 규칙 수정 절차

1. `#제안`으로 규칙 변경 제안
2. 커뮤니티 투표 진행
3. 통과 시 CommunityRule 업데이트
4. 변경된 규칙 공지

## 5. 자주 묻는 질문

### Q: 투표 기간을 변경할 수 있나요?
A: 네, 긴급한 사안은 방장이 투표 기간을 조정할 수 있습니다. 이 자체도 제안으로 올릴 수 있습니다.

### Q: 내 제안이 부결되면 다시 제안할 수 있나요?
A: 네, 하지만 내용을 수정하거나 보완한 후 다시 제안하는 것을 권장합니다.

### Q: 투표를 취소할 수 있나요?
A: 제안자가 요청하면 방장이 투표를 취소할 수 있습니다 (텔레그램은 자동 진행).

### Q: 익명 투표는 안 되나요?
A: 민주적 투명성을 위해 공개 투표를 기본으로 합니다. 필요 시 제안으로 변경 가능합니다.
