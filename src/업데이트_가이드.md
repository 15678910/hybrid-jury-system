# 🎨 배경 흐림 효과 업데이트

## ✅ 변경사항

### 1. 모달 배경 개선
**변경 전:**
```javascript
bg-black bg-opacity-90  // 90% 불투명 검정 (거의 안 보임)
```

**변경 후:**
```javascript
bg-black bg-opacity-50 backdrop-blur-sm  // 50% 투명 + 흐림 효과
```

**효과:**
- ✨ 뒤 페이지가 흐릿하게 보입니다
- ✨ 포스터에 더 집중되는 효과
- ✨ 더 세련된 UI

## 🎵 음악 문제 해결

### 확인사항 1: 파일명 확인
Windows 탐색기에서:
1. `보기` → `표시` → `파일 이름 확장명` 체크
2. public 폴더에서 음악 파일 이름 확인
3. 정확히 `시민법정_참심제_reggae1.mp3` 인지 확인

### 확인사항 2: 자동재생 버튼
모달을 열었을 때 화면에 다음 버튼이 보이나요?
```
🎵 음악 재생하기
```

이 버튼이 보이면:
- 브라우저가 자동재생을 차단한 것입니다
- 사용자가 직접 버튼을 클릭해야 합니다
- 이것은 정상입니다!

### 확인사항 3: 브라우저 콘솔
1. F12 → Console 탭
2. 에러 메시지가 있는지 확인
3. "404" 에러가 있다면 파일명이 다른 것입니다
