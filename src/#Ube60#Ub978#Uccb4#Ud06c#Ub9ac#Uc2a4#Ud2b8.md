# ⚡ 빠른 배포 체크리스트

## 🎯 5분 안에 배포하기

### 1단계: 파일 다운로드 및 교체 (2분)
```
□ App.jsx 다운로드
□ Poster.jsx 다운로드
□ src/App.jsx 파일 교체
□ src/Poster.jsx 파일 교체
```

### 2단계: 필수 파일 확인 (1분)
```
□ public/참심제_웹자보qrcode.png 파일 존재 확인
□ public/시민법정_참심제_reggae1.mp3 파일 존재 확인
```

### 3단계: 테스트 (1분)
```bash
npm run dev
```
```
□ http://localhost:5173 접속
□ "🎵 포스터 보기" 버튼 보임
□ 버튼 클릭 → 모달 열림
□ X 버튼 → 모달 닫힘
□ "지금 참여하기" → 스크롤 작동
```

### 4단계: 빌드 및 배포 (1분)
```bash
npm run build
```
```
□ dist 폴더 생성 확인
□ 호스팅 서버에 배포
□ 라이브 사이트에서 재확인
```

---

## 🚨 문제 발생 시 빠른 해결

### "🎵 포스터 보기" 버튼이 안 보임
```
1. Ctrl + Shift + R (강력 새로고침)
2. src/App.jsx 595번째 줄 확인
3. npm run dev 재시작
```

### 모달이 열리지 않음
```
1. F12 → Console 탭 확인
2. 에러 메시지 확인
3. Poster.jsx 교체 확인
```

### 이미지가 안 보임
```
1. public 폴더 확인
2. 파일명 확인 (한글)
3. 파일 경로 확인
```
