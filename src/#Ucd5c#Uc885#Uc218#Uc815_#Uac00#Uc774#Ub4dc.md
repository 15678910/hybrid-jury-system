## 🎵 음악 자동 재생 트릭

```javascript
// 음소거로 먼저 시작
audioRef.current.muted = true
await audioRef.current.play()

// 성공하면 음소거 해제
audioRef.current.muted = false
```

이 방법으로 대부분의 브라우저에서 자동 재생 성공!
