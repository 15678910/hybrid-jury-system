# 🎯 SNS 자동 팝업 기능 완성!

## ✅ 구현 완료!

이제 SNS에서 특별한 링크를 클릭하면 → 홈페이지가 열리면서 → 자동으로 포스터 팝업이 뜹니다!

---

## 🔗 SNS에 올릴 링크 (2가지 중 선택)

### 방법 1: ?popup=true
```
https://시민법정.kr?popup=true
```

### 방법 2: #poster (해시태그)
```
https://시민법정.kr#poster
```

**둘 다 작동합니다!** 편한 것을 사용하세요.


## 🎨 추가 기능 (현재 구현됨)

### ✅ 흐릿한 배경
- 팝업 뒤 메인 페이지가 흐릿하게 보임
- 포스터에 집중 효과

- "💪 지금 참여하기" 버튼 → 자동으로 서명 섹션 이동

## 🎯 핵심 변경사항

### App.jsx (60-66번째 줄)
```javascript
// URL 파라미터 확인하여 포스터 모달 자동 열기
useEffect(() => {
    const urlParams = new URLSearchParams(window.location.search);
    if (urlParams.get('popup') === 'true' || window.location.hash === '#poster') {
        setShowPosterModal(true);
    }
}, []);
```

**작동 원리:**
1. 페이지 로드 시 URL 확인
2. `?popup=true` 또는 `#poster` 발견
3. `setShowPosterModal(true)` 실행 → 팝업 자동 열림!

## 💡 Pro Tips

### Tip 1: 짧은 URL 만들기
긴 URL이 싫다면 단축 URL 서비스 사용:
- Bitly: `bit.ly/참심제팝업`
- 네이버: `me2.do/xxxxx`

### Tip 2: QR 코드 생성
오프라인 포스터에 QR 코드 추가:
```
https://시민법정.kr?popup=true
```
→ QR 코드 생성 → 인쇄

### Tip 3: 다양한 진입점
```

https://시민법정.kr?popup=true&source=instagram
```
어디서 유입되는지 추적 가능 (나중에 분석 추가 시)

## 🚨 주의사항

### 1. 브라우저 자동재생 정책
- 대부분의 브라우저는 자동재생을 차단합니다
- "🎵 음악 재생하기" 버튼이 나타나는 것이 정상입니다
- 사용자가 버튼을 클릭하면 음악이 재생됩니다

### 2. 모바일 고려
- 모바일에서도 팝업이 잘 보입니다
- 작은 화면에서도 포스터가 선명합니다

### 3. 뒤로가기 버튼
- 팝업 닫으면 URL 파라미터는 유지됩니다
- 새로고침하면 다시 팝업이 열립니다
- 이것이 의도된 동작입니다!
