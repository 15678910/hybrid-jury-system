### **4. 법률안 내용 업데이트 (6개 → 8개 항목)**

**추가된 항목:**

**6. 임기 (제14조)**
```
사건별 선임. (독일/스웨덴식 장기 임기 아님). 
국민 참여 기회를 극대화하고 부담 감소.
```

**8. 절차 (제23조)**
```
집중심리 및 연일 개정 권장. 재판장의 이해돕기 설명. 
재판절차 이해 가능하게 국민 참여 보장.
```

---

### **5. 기존 항목 내용 수정**

**1. 목적 (제1조)**
```
국민이 참심법관으로 참여, 직업법관과 함께 재판권을 행사. 
사법의 민주적 정당성, 투명성, 신뢰 증진을 목적으로 함.
```

**2. 적용 범위 (제3조)**
```
1심 중대 형사사건 (예: 법정합의부 사건, 사형, 무기 또는 단기 1년 이상 징역 사건). 
피고인 자백 여부와 무관.
```

**3. 재판부 구성 (제5조)**
```
직업법관 3인 + 참심법관 6인 (필요시 9인까지 증가 가능). 
참심원이 다수를 구성.
```

**4. 참심법관 권한 (제6조)**
```
직업법관과 동등한 권한 (질문권, 평의, 표결권). 
사실인정, 법령 적용, 유/무죄 및 양형에 모두 참여.
```

**5. 선발 (제8조~제12조)**
```
무작위 추출 + 심사 + 기피 절차. 
만 25세 이상 국민 명부에서 선정. 
스웨덴식 정당 추천 배제.
```

**7. 평결 (제25조)**
```
특별다수결. 과반수 찬성 + 유죄 판결 시 
'직업법관 1명 이상' 및 '참심법관 1명 이상' 찬성 모두 요구.
```
